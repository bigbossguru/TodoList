# TodoList
A simple TODO list

### Install Python version 3.8.5

### Activate virtual envoriament
```
python -m venv venv
```
```
source venv/Scripts/activate
```
### Install packages
```
pip install -r requirements.txt
```
### Update db in django
```
python manage.py migrate
```
### Run server
```
python manage.py runserver
```
